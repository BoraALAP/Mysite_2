TSA - Toronto School of art

Toronto school of art is a art school that teaches to all levels. Purpose of this new website was making purchase avaiable without using any third party website. So this gave us a creative opportunity for building a whole new website.

When the page loads all the elements takes its place one by one. TSA writing and the painting behind it moves when the cursor moves on the page. It basically tracks down the users movement on the homepage.

On the courses page, side navigation is fixed and it stays with the user all the time. It helps users to navigate through between the filtered courses. User can choose to see which course is avaiable for the which term or type of the courses.

Side navigation works the same vay on this page. It helps user to navigate between the sections. Also, we added a filterin system on the top of the sections. So if user wants to filter it for level of the course or duration. They dont need to go back to the previous page to filter it.

This page is for the single course page. User can pick the time frame they want to choose from, and see what they will learn.

Cart page will let them put their name, Also, this page will allow them to remove a course or add someone else to this course.

They can also see their total, as well as put the promo code if they have any.

Welcoming section has a cinematography as a background. User can access to their account direcly from this section, and if they want to access to menu, they can access it from the menu button at the top.

When they start scrolling navigation gets a background and sticks to top of the page. So, its always there to help.

Meridian is a company that values their community and customers, so instead of filling the sections with a lot of information about the products or services we went with the news about the community on the homepage.

Third concept uses the fixed, full height, left navigation style. Left nav has the full navigation opten still in it. This kind of combines first and second concepts style.
